How this website is created

Welcome to my first blog. In this blog I am going to write about how I made my personal website

Backend

After doing some research about Python backend framework, I ran into the two most popular framework: Django and Flask. Where Django is a full-stack framework, Flask is a micro and lightweight web framework. After trying out both of them, I figured to go with Flask since it has a shorter learning gap for a starting project.

This server is hosted on a free service tier on Heroku

Frontend

I used simple HTML, CSS, Boostrap to power the frontend of this website.

Additionally, I utilized Javascript to make this website more interesting and fun for the users.

  1. Everytime you visit my website, the background image is a random dog picture fetched from a Dog Pic API
