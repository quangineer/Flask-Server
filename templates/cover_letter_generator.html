Cover Letter Generator

Writing a Cover Letter for each job application is no longer a hassle with this Cover Letter Generator

Posted by Quangineer on June 28, 2019

The need of a Cover Letter Generator

For most of the time we apply for an online job we are required to send in a Cover Letter. With the huge amount of applications we send out each day comes with unavoiable and painful mistakes in our applications, whether it is Date, Company Name or simple typos.

The Cover Letter Generator solves all these problems. It is made to produce an erorr free cover letter while being flexbile enough so that you can modify the content of the file.

Here is the Github Link

Installation

The only thing we need for this project is the python.docx library to be able to manipulate word document.

How does it work?

This project follows a simple structure of a typical cover letter:

  • Contact Info
  • Self Introduction and background
  • Projects I have done
  • Closing and Thank you

Personal information is stored in a separate json file and the user just pick the right information when they make a cover letter.
